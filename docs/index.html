Maxy documentation

Your personal AI on your Mac: voice in the terminal or a Telegram bot. Powered by Gemini and local Ollama.

Developed and designed by Gokulakrishnan

Overview

Maxy runs two interfaces from one codebase: a push-to-talk or wake-word voice assistant (Whisper + macOS say), and a Telegram bot with Gmail, weather, search, reminders, todos, and notes. Conversation and memory live in SQLite under your home directory.

Install

Requires Node.js 16+ and Python 3.8+. macOS is the primary target; Linux is supported; Windows is untested.

npm: npmjs.com/package/maxyy — package name is maxyy (two ys), not maxxy. 

npm i -g maxyy

Alternatives: install from GitHub — npm i -g git+https://github.com/Gokulakrishnanxn/maxy.git — or clone and npm link, or run Python only (see Manual setup).

Quick start

# 1. First-time setup (API keys, voice, model)
maxyy setup

# 2. Pick your interface
maxyy voice          # voice assistant in terminal
maxyy telegram       # Telegram bot

AI models

Gemini (default)

Cloud, fast, multilingual. Model: gemini-2.5-flash. Set GEMINI_API_KEY in ~/maxy/.env.

Ollama (local)

Private, no API key. Install Ollama and run ollama serve. Use any pulled model.

Switch in Telegram

/model                  # current model
/model gemini           # Gemini
/model llama3.1:8b      # Ollama model
/model list             # local Ollama models
Auto-fallback: If Gemini quota is exceeded, Maxy switches to your best Ollama model and notifies you. If Ollama is down, it can fall back to Gemini when available.

Features

Gmail

Unread mail via /inbox or natural language. Sending requires your confirmation. One-time setup: Google Cloud project, Gmail API, credentials.json in ~/maxy/, then browser OAuth.

Weather

Powered by wttr.in — no API key.

Web search

DuckDuckGo via /search or natural requests.

Reminders

/remind 30m take a break
/remind 2h call mom
/remind 1d submit invoice
/reminders

Units: s seconds, m minutes, h hours, d days.

Todo

/todo add finish the report
/todo list
/todo done 3
/todo delete 3

Notes & memory

/note Priya's birthday is March 12

Facts like "my name is…", "I work at…", "remember that…" can be stored automatically.

Configuration

Config file: ~/maxy/.env (create with maxyy setup or edit by hand).

GEMINI_API_KEY=your_key_here
TELEGRAM_BOT_TOKEN=your_token_here
MAXY_VOICE=Samantha
MAXY_BACKEND=gemini
MAXY_OLLAMA_MODEL=llama3.1:8b
VOICE_USER_ID=voice_local

Override data directory:

MAXY_HOME=/custom/path maxyy voice

Data & privacy

Data Location
Chats, notes, tasks~/maxy/maxy.db
Gmail OAuth~/maxy/credentials.json, tokens beside it
API keys~/maxy/.env

With the Ollama backend, inference stays local. Gemini sends prompts to Google’s API — use Ollama when you want minimal cloud use.

Manual setup (without npm)

git clone https://github.com/Gokulakrishnanxn/maxy.git
cd maxy
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

Create ~/maxy/.env (or copy from project examples) with your keys, place Gmail credentials.json if needed, then:

python voice.py    # voice
python main.py     # Telegram
